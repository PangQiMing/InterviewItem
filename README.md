# 启动项目步骤

### 1.安装ffmpeg步骤
前往ffmpeg官网,下载ffmpeg压缩包
```text
https://ffmpeg.org/download.html
```
把压缩包解压出来,配置好系统环境变量,在终端输入
```text
ffmpeg -version
```
出现ffmpeg版本信息就说明安装成功了

### 2.启动项目
mysql数据库的连接,打开.env文件可以修改连接数据的配置

最后启动项目
```text
go run main.go
```
